# Smart Task Organizer (SE Lab)

A simple task management API built with Flask.

## Run
pip install flask
python app.py

## Endpoints
- GET /tasks
- POST /tasks
- PUT /tasks/<index>
- DELETE /tasks/<index>
- GET /tasks/<index>/complete
- GET /tasks/sort/priority
- GET /tasks/sort/deadline
- GET /tasks/filter/completed
- GET /tasks/filter/not_completed
- GET /tasks/filter/high_priority
- GET /tasks/export
